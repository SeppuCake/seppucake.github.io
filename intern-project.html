About the Project

CompuRex Corporation is a company that provides Software as a Service (SaaS). Delivering new technological solutions and digital media contents to organizations. I am delighted for the opportunity to work with this company with the amount of projects and tasks each employee responsible to deliver. Through this project, I was tasked to create a responsive website and a dashboard for statistics displays such as number of orders, number of procurements, and etcetera. The project also required me to create a database included with the Entity Relationship Diagram (ERD) for each modules such as:

  • User Login Module
  • Dashboard
  • User Access Control Module (Admin Use)
  • Product Registration Module
  • Procurement Registration Module
  • Messaging Module
  • Create, Read, Update, Delete (CRUD) Modules (For Product & Procurement Modules)
