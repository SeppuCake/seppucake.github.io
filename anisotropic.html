About the Project

This is one of the freelance work I was hired to partake in for a local student. The client provided the info of their project and required an assistant and a teacher to create a Java program for their project in order to proof a concept in Anisotropic Diffusion. It was quite tough deciphering the formula and I had to process the equation that was in their paperwork and translate it to Java language to make the program based on Perona-Malik, Conductance Estimation Methods and BLACK et al. The principles applied for the program to work:

  • Persona-Malik
  • Conductance Equation
  • Black et al
